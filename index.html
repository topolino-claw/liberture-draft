What is Liberture?

A skill library that plugs into OpenClaw. Focused on human optimization — sleep, nutrition, exercise, cognition, mind, and finance.

✅ What it is

A community skill library. OpenClaw-compatible tools organized by pillar. Templates for creating your own. Copy-paste integration.

❌ What it's not

Not an app. Not a tracking platform. Not replacing OpenClaw. It's symbiotic — skills that make your agent smarter about health.

Create a Skill

Contributing is simple. Follow the template, submit a PR.

1

Fork the repo

Clone liberture/skills and create a new folder for your skill.

2

Use the template

Copy _template/ and fill in SKILL.md with your interface spec.

3

Implement & test

Write the logic. Test it with your own OpenClaw agent.

4

Submit PR

We review, tag by pillar, and merge. Your skill goes live.

# Quick start git clone https://github.com/liberture/skills cd skills cp -r _template my-new-skill # Edit SKILL.md, implement, test, PR

Integrate with OpenClaw

Using a Liberture skill is copy-paste simple.

# Option 1: Copy to your skills folder cp -r liberture/skills/sleep-cycle-calculator ~/.openclaw/skills/ # Option 2: Clone the whole library git clone https://github.com/liberture/skills ~/.openclaw/liberture-skills # Your agent now has access to all Liberture skills

Skills follow OpenClaw's SKILL.md format. Your agent reads the spec, knows the interface, and can call the tool.
