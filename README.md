## Vision

Liberture is evolving from "Biological Operating System" to an **open-source platform for building AI systems focused on human performance**.

Three pillars:
1. **Knowledge** — Curated content on performance domains
2. **Functions** — Agent skills and interactive tools
3. **Agents** — AI systems that use the knowledge + functions

See `SYSTEM_DESIGN.md` for full architecture.
